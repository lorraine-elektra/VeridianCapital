# Veridian Capital Partners
A premium single-page marketing and investor-relations website for Veridian Capital Partners, built to connect global capital with high-impact opportunities across Africa and emerging markets.
Built as a fully static, framework-free site using a single `index.html` with inline HTML, CSS, and Vanilla JavaScript.

## Live Demo
https://veridiancapital.pages.dev/


## Features
* Responsive single-page design
* Zero build step or dependencies
* Smooth-scroll navigation
* Interactive animations and counters
* Investor enquiry and contact forms
* SEO and Open Graph ready
* Accessibility-focused implementation


## Tech Stack
* HTML5
* CSS
* Vanilla JavaScript (ES2017+)
* SVG + Canvas API
